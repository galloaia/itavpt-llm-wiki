# Wiki Filo Studio — vault Obsidian

Vault generato dall'esplosione di `FILO_WIKI.md` in note atomiche collegate.

## Apri in Obsidian
1. Obsidian → *Open folder as vault* → seleziona questa cartella.
2. Apri `Home.md`, attiva la **Graph view**: i concetti sono hub, sessioni/prompt/fonti vi orbitano.

## Due usi, una sorgente
- **Per te (umano):** navighi in Obsidian — grafo, ricerca, tag, backlink.
- **Per gli LLM:** `python3 compile.py` rigenera `FILO_WIKI_COMPILED.md`, il file unico da incollare/fetchare.

## Stato
Completo end-to-end: tutte e 4 le sessioni (S1–S4) nel diario; libreria prompt completa (incluse S3 e S4); glossario atomizzato in note-concetto; 7 fonti esterne ricercate, sintetizzate e linkate.

## Note
- I nomi file usano accenti dove serve; Obsidian e Linux li gestiscono. Rinominando una nota in Obsidian, i link si aggiornano da soli.
- I wikilink automatici sono un primo strato (per alias): solido come scheletro, arricchibile a mano dove il senso lo merita.
